# App for managing your contacts

# In development

## About

- Spring boot
- [Quasar framework - VueJS](https://quasar.dev/)


## Quasar project

    Install dependencies

        npm install

    Run project

        quasar dev

## Spring boot

    mvn spring-boot:run
